不可夸大结论

  • 可以声称真实 A100 runtime label provenance check 已完成并生成脱敏公开摘要。
  • 不能声称 checkpoint release、adapter release、held-out generalization、production readiness 或 live-browser improvement。
  • 不能把 assistant_tokens_carry_loss=true 解释成模型已学会 Browser Task Contract；它只说明目标函数路径里 assistant tokens 参与 loss。
  • prompt_tokens_masked=false 是一个重要限制：当前证据不支持 assistant-only loss-mask claim。
