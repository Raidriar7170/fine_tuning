本阶段改了什么

  • 新增 voice2task-eval diagnose-alignment，比较 gold target 和 raw prediction 字段。
  • 新增 alignment diagnostics：每条 mismatch 记录 row id、field path、mismatch category、gold value summary、prediction value summary。
  • 报告 schema-invalid prediction 时不修复、不 coerce、不 normalize，也不把它算作 valid contract。
  • 补强公共安全脱敏：diagnostic row id 和字段摘要都会过滤私有路径、私有 IP 与 secret-like 字符串。
  • 同步 contract-evaluation spec，加入 target-prediction alignment mismatch requirement。
