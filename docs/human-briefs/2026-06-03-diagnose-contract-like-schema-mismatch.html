本阶段改了什么

  • 新增 voice2task-eval diagnose-schema，从 gold/prediction pair 生成 JSON 和 Markdown schema diagnostics。
  • 新增字段级诊断：缺必填字段、无效枚举、无效类型、空必填字符串、固定值不匹配。
  • 生成 post-recovery 诊断报告，并保留 invalid predictions 仍然 invalid 的边界。
  • 同步 contract-evaluation spec，记录 contract-like schema mismatch 诊断要求。

不要夸大的结论

这不是 checkpoint release、adapter release、模型质量提升、production readiness、full-private-corpus release，也不是 live-browser benchmark improvement。它只说明当前 private-adapter public-sample 输出已经有 contract-like 外壳，但仍未满足 Browser Task Contract schema。
