extract-price canonical wording 阶段简报

一句话结论：7B A100 训练在 bounded public train split 上把 extract-price strict residual 消掉了：contract_exact_match=1.0000，extract-price 3/3 strict exact。

当前状态

阶段状态：archived；证据类型：public-sample train-split diagnostic；不是 dev/test 泛化、生产可用、checkpoint/adapter release 或 live-browser benchmark 结论。

本阶段改变

  • 新增 extract-price canonical wording policy：多少钱/标价/页面上的商品价格 必须落到 slots.target="商品价格" 与 normalized_command="提取页面商品价格"。
  • 新增 3 类 DPO hard negative：generic 价格、listed 标价、extra-particle 提取页面上的商品价格。
  • public DPO 从 48 对更新到 51 对；A100 7B train-split rerun 后只回传 sanitized evidence。
  • 修复 prediction runtime 的 markdown fence suppression：只传单 token bad words，绕开 transformers 5.5.0 multi-token CUDA bug。
