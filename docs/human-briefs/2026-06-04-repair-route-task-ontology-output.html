本阶段改变

  • SYSTEM_PROMPT 新增 route ontology：route 是 Browser Task Contract execution channel enum。
  • 明确 weather、shopping、email、media、URL host 等领域词不能作为 route，应进入 task_type、slots 或 normalized_command。
  • 新增 weather 示例：天气查询使用 task_type=search 和 route=search_web。
  • prompt_constraint_summary() 新增三个可测试 flag，prediction dry-run metadata 会带出这些约束。
  • 新增本地 public-safe evidence pack，记录 prior A100 failure 与本阶段边界。
